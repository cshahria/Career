How to Manually Add New Tiles

Follow these two steps to scale your custom dashboard layout without changing any core JavaScript logic.

Adding a completely new category tile requires copying and pasting code snippets into exactly two places inside your index.html file.

Step A: Add the Visual Tile Button

Scroll down to the <div class="tile-grid"> section of your file. Paste a new tile code block right below the existing ones. Make sure to choose a unique modal name for the openModal('...') trigger.

<!-- Duplicate this block inside the .tile-grid container -->
<div class="tile" onclick="openModal('yourNewModal')">
  <div class="icon">🔮</div> <!-- Replace with any emoji -->
  <div class="name">New Category</div> <!-- Label text -->
</div>

Step B: Add the Form Pop-up Window (Modal)

Scroll down to the bottom of the document and look for the other modal windows. Paste this new template block right before the final closing </body> tag. Ensure the container id="..." matches your modal name from Step A exactly.

<!-- MODAL: NEW CATEGORY TRACKER -->
<div id="yourNewModal" class="modal-overlay">
  <div class="modal-content">
    <button class="modal-close" onclick="closeModal('yourNewModal')">×</button>
    <h3>Log New Entry</h3>
    
    <!-- 1. Update form action URL with your Google Form response endpoint -->
    <form method="POST" action="https://docs.google.com/forms/d/e/YOUR_NEW_FORM_ID/formResponse" target="hidden_iframe" onsubmit="handleFormSubmit()">
      
      <!-- 2. Duplicate or adjust these form-group fields to match your spreadsheet inputs -->
      <div class="form-group">
        <label>Field Name</label>
        <input type="text" name="entry.123456789" placeholder="Hint text..." required>
<button type="submit" class="submit-btn">Save Entry</button> </form> </div> </div>

Quick Reminders

  1. Required Attribute: Adding required inside your <input> or <textarea> tags forces the browser to prevent submission if the field is empty.
  2. Field Mapping: Always pull the exact entry.####### data keys directly from your target Google Form's live page source so submissions line up perfectly with your sheet columns.
